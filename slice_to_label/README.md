# Slice to Label

AI-powered video slice labeling tool with dual-stage analysis mechanism.

## Features

- Automated video slice analysis
- AI-powered label generation
- Dual-stage analysis mechanism
- Support for multiple AI models (Google Gen AI, DeepSeek)

## Installation

```bash
uv sync
```

## Usage

```bash
python batch_slice_to_label.py
```

## Configuration

Edit the configuration files in the `config/` directory to set up your API keys and preferences.

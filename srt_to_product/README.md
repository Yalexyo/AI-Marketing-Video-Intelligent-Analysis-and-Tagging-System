# SRT to Product

AI-powered SRT to product video generation using DeepSeek API.

## Features

- Convert SRT subtitle files to product videos
- AI-powered content generation
- Video processing with FFmpeg
- Support for DeepSeek API
- Batch processing capabilities

## Installation

```bash
uv sync
```

## Usage

```bash
python run.py
```

## Configuration

Set up your environment variables:
- `DEEPSEEK_API_KEY`: Your DeepSeek API key
- Configure other settings in the `config/` directory
